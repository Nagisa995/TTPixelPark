Тестовые задания
Задание на junior веб-разработчика
Написать приложение, которое будет получать данные о заказах из Pixlpark через API и выводить их на странице.

Pixlpark API:
точка входа: http://api.pixlpark.com/
получение списка заказов: http://api.pixlpark.com/orders

Разрешается использовать любые клиентские фреймворки, а также проксировать запросы через бекенд (backend).
